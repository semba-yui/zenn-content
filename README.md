# zenn-content

## コマンド集

https://zenn.dev/zenn/articles/zenn-cli-guide

### コンテンツ管理用のディレクトリを作成. 初回のみ実行

```shell
zenn init
```

### コンテンツをブラウザでプレビュー

```shell
zenn preview        
```

### 新しい記事を追加

```shell
zenn new:article
```

### 新しい本を追加

```shell
zenn new:book
```

### 記事の一覧を表示

```shell
zenn list:articles
```

### 本の一覧を表示

```shell
zenn list:books
```

### zenn-cliのバージョンを表示

```shell
zenn --version
```

```shell
zenn -v
```

### ヘルプ

```shell
zenn --help
```

```shell
zenn -h
```
